# IoT-based-Food-Quality-Monitoring

An IoT-based Food Quality Monitoring System that uses gas sensors, environmental sensors, and pH analysis to detect food spoilage in real time, with support for AI-based prediction. <br>

Key Features <br>
- Real-time monitoring <br>
- Gas-based spoilage detection <br>
- pH and environmental analysis <br>
- Cloud integration <br>
- AI-based prediction (ongoing) <br>

System Overview <br>
Sensors: MQ135, MQ136, MQ137, DHT22, pH <br>
Microcontroller: ESP32  <br>
Cloud: ThingSpeak <br>

How It Works <br>
Sensors → ESP32 → Cloud → Analysis → Alert <br>
